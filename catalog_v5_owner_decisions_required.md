# Catalog V5 Owner Decisions Required

This file records the current V5 scope decisions after owner confirmation on 2026-07-03.

## Decisions applied into V5 public catalogs

| # | Item | Applied V5 treatment |
|---:|---|---|
| 1 | Ceiling material | Ceiling is specified as plaster ceiling board. Basic uses plain flat plaster ceiling board; Standard uses flat plaster ceiling board with smoother finish/access-panel allowance where required; Better Finish may use simple design plaster ceiling such as tray/L-box/feature ceiling when listed. |
| 2 | Tile skirting | Tile skirting is included up to about 4 inch / 100mm to new plastered wall areas, except wall areas already tiled, blocked by cabinet/table top, or otherwise stated. |
| 3 | Door allowance | Normal enclosed extension may include one standard single-leaf door set up to approx. 3ft x 7ft when required by layout. Sliding, folding, full-height, aluminium/glass feature door and oversized openings remain quote-only. |
| 4 | Window allowance | Normal enclosed extension may include one aluminium framed window: Basic up to approx. 4ft x 3ft, Standard up to approx. 5ft x 4ft, Better Finish up to approx. 6ft x 4ft, subject to actual quotation. |
| 5 | Electrical quantities | For normal extension area up to about 150 sqft: Basic includes up to 1 light point + 2 sockets, Standard up to 2 light points + 3 sockets, Better Finish up to 3 light/downlight points + 4 sockets, all subject to practical route and existing suitable circuit. |
| 6 | Plumbing quantities | For back/kitchen/yard/laundry extension only: Basic includes up to 1 cold-water point + 1 waste outlet + 1 floor trap if route is practical; Standard adds washing-machine point or outdoor tap where listed; Better Finish adds one utility-sink or outdoor-tap point where listed. |
| 7 | WC package tier | Basic bathroom uses 1 standard two-piece close-coupled WC set. Standard uses 1 better-grade two-piece close-coupled WC set. Better Finish uses 1 premium one-piece WC combo set where available within the selected allowance. Brand/model must be listed. |
| 8 | Toilet door | Bathroom add-on includes 1 toilet door set up to approx. 2.5ft x 7ft, typically PVC or aluminium/PVC type by quotation, including frame, hinges and lockset. |
| 9 | Bathroom tile height | Wall tile is to wet-wall/agreed height. Better Finish may be agreed height or near/full height where listed. Exact wall height must be stated in quotation. |
| 10 | Sink and tap | Wet Kitchen Set A/B/C may include 1 standard kitchen sink and 1 standard kitchen tap when listed. Kitchen Table Top Only may exclude sink/tap unless listed. Premium sink/tap remains separately quoted. |
| 11 | Wet-kitchen cabinet | Wet Kitchen Set A/B/C means concrete/tiled table top and wet-kitchen preparation only. Base cabinet, wall cabinet and aluminium cabinet remain quote-only unless specifically listed. |
| 12 | Paint system | Internal surfaces include surface preparation, sealer/primer where required and two coats of emulsion paint. Exterior exposed new walls use exterior-suitable paint. Better Finish may use premium paint range where listed. |
