## Required first checks

1. List all PDF/drawing files in `Drawing/` without modifying them.
2. Confirm drawing title, revision/date if visible, and sheet count.
3. Identify measurable scopes before quantity calculation:
   - roof / awning / metal works;
   - ceiling / board / strip ceiling;
   - wall / partition / cladding;
   - door / window / glass;
   - gutter / flashing / drainage;
   - electrical / lighting if shown;
   - floor / finishes if shown;
   - other items shown in drawings.
4. Record every assumption separately before BOQ pricing.
5. Flag missing dimensions or unclear details instead of guessing silently.
