# Kitchen Drawing Correction V1

## Purpose

Owner corrected the kitchen size basis from the drawing for the original ActiveBOQ customer job.

The previous estimate used a small typical allowance for kitchen cabinet. That is too low after owner pointed out the actual drawing dimensions and house type.

## Owner correction

House type:

`Rumah berkembar / semi-detached`

Kitchen usable wall dimensions from drawing:

| Wall / run | Dimension |
|---|---:|
| Back wall | 3962 mm |
| Side wall / usable side run | 5485 mm |
| Door deduction | 3 ft |

## Calculation

Convert to feet:

| Item | Formula | Result |
|---|---|---:|
| Back wall | 3962 / 304.8 | 13.00 ft |
| Side wall | 5485 / 304.8 | 18.00 ft |
| Combined L-shape before door deduction | 13.00 + 18.00 | 31.00 ft |
| Less door allowance | 3.00 ft | 3.00 ft |
| Maximum practical L-shape base/countertop run | 31.00 - 3.00 | 28.00 ft |

## Revised kitchen takeoff basis

Use this as the corrected provisional kitchen basis:

| Item | Previous estimate | Corrected estimate | Reason |
|---|---:|---:|---|
| Base cabinet blockboard | 10 ft | 28 ft | L-shape maximum available run from drawing less door allowance. |
| Wall cabinet melamine | 8 ft | 18 ft | Logical upper cabinet allowance for larger semi-D kitchen; final shop drawing can adjust. |
| Porcelain/sintered countertop | 10 ft | 28 ft | Countertop follows base cabinet L-shape run. |
| Quartz backsplash | 25 sqft | 70 sqft | 28 ft x 2.5 ft backsplash height allowance. |
| Minor kitchen arch | 1 set | 1 set | No change. |

## Important note

The corrected 28 ft is a maximum practical L-shape run based on the drawing and owner correction. Final measurement and cabinet shop drawing still control the final quantity.

Do not continue using the earlier 10 ft kitchen-base / countertop assumption for this customer estimate.
