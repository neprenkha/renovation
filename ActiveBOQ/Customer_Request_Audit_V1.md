## Catalog coverage decision

This is mostly a custom interior renovation / fit-out job, not a ready house-extension package.

Use catalog V5 only for general customer-facing wording and matching scope principles such as plaster ceiling, electrical point explanation, plumbing relocation caution, paint and exclusions.

Do not force the whole request into single-storey/double-storey extension catalog package pricing.

Use manual BOQ workflow from `README - BOQ.txt` for measured pricing because the request includes custom cabinetry, ceiling, wiring, plumbing relocation, mirror/fluted/wainscoting and SPC flooring.

## Required working method

1. Keep `Customer Request.txt` unchanged as source evidence.
2. Create new upgrade files only for audit, SPEC, takeoff and quotation.
3. Start with SPEC A:C rows before filling any BOQ/takeoff blocks.
4. Split scopes by practical site sequence and trade package.
5. Use exact Materials/Labour master-list descriptions where available.
6. If exact materials/labour items are missing from master lists, mark `INPUT NEEDED` or create a master-list update proposal instead of inventing names silently.
7. Do not show internal raw costing or supplier names in customer-facing quotation.

## Missing information before reliable pricing

These details are required before a firm quotation:

1. Floor plan/drawing file name and readable dimensions.
2. House type and floor level involved.
3. Kitchen cabinet length: base cabinet run, wall cabinet run, tall unit if any and island if any.
4. Countertop length/depth and backsplash area/height.
5. TV cabinet width/height/depth and arch detail.
6. Master bedroom wardrobe/cabinet length/height/depth and make-up table size.
7. Shoe cabinet width/height/depth.
8. Living hall ceiling C-box length/area and kitchen flat ceiling area.
9. Paint wall/ceiling area or room list.
10. Wiring point count detail: number of downlights/eyeballs, plug points, switches, LED driver points and air-cond point route.
11. Plumbing relocation distance and whether floor/wall hacking is required.
12. Mirror/fluted/wainscoting dimensions and material preference.
13. SPC flooring living hall area and floor preparation condition.
