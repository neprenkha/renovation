# Ceiling Dimension Correction V1

## Purpose

Owner asked whether the kitchen flat ceiling and living C-box ceiling were already calculated using the latest drawing dimensions.

Answer: not fully. The previous estimate still used rough quantities:

- living C-box: 80 rft;
- kitchen flat ceiling: 120 sqft.

This correction uses the owner-provided drawing dimensions.

## Corrected dimensions

| Area | Dimension |
|---|---:|
| Kitchen flat ceiling | 6096mm x 5792mm |
| Living C-box ceiling | 9755mm x 6096mm |

## Kitchen flat ceiling calculation

| Item | Formula | Result |
|---|---|---:|
| Length | 6096mm / 304.8 | 20.00ft |
| Width | 5792mm / 304.8 | 19.00ft |
| Area | 20.00ft x 19.00ft | 380.05sqft |
| Rate used | RM5.00/sqft | RM5.00/sqft |
| Amount | 380.05 x RM5.00 | RM1,900.26 |

Previous estimate:

`120sqft x RM5.00 = RM600.00`

Increase:

`RM1,900.26 - RM600.00 = RM1,300.26`

## Living C-box ceiling calculation

Use perimeter of living hall rectangle for C-box:

| Item | Formula | Result |
|---|---|---:|
| Length | 9755mm / 304.8 | 32.00ft |
| Width | 6096mm / 304.8 | 20.00ft |
| C-box perimeter | 2 x (32.00 + 20.00) | 104.01rft |
| Rate used | RM18.00/rft | RM18.00/rft |
| Amount | 104.01 x RM18.00 | RM1,872.17 |

Previous estimate:

`80rft x RM18.00 = RM1,440.00`

Increase:

`RM1,872.17 - RM1,440.00 = RM432.17`

## Revised ceiling subtotal

| Ceiling item | Previous amount | Revised amount | Increase |
|---|---:|---:|---:|
| Living C-box | RM1,440.00 | RM1,872.17 | RM432.17 |
| Kitchen flat ceiling | RM600.00 | RM1,900.26 | RM1,300.26 |
