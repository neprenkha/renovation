# 1044sqft Flooring Calculation Template V1

## Purpose

This document is the standard calculation template for flooring materials under:

`MISC/Nii Flooring SPC, Vinyl/`

It applies to:

- Carpet;
- Vinyl;
- SPC;
- foam / underlay;
- glue;
- profile / trims;
- skirting;
- other stock-size flooring accessories.

This is a working-cost template, not a customer selling price unless markup is added.

## Core rule

Do not calculate stock materials by loose tiny quantity when supplier sells by full stock unit.

Use:

- full box;
- full roll;
- full piece;
- full tub;
- full set;
- actual supplier stock size shown in the product folder or price list.

## Example input

Customer gives:

`1044 sqft`

No detailed room dimensions are given.

Labour example:

`RM1.20/sqft`

## Square-root method for quick perimeter

| Formula | Result |
|---|---:|
| Side = sqrt(1044 sqft) | 32.31 ft |
| Four-side perimeter = 32.31 x 4 | 129.24 ft |

Use this only when exact room perimeter is not available. If floor plan has actual dimensions, use actual perimeter instead.

## Floor material quantity

Apply 10% wastage to floor area:

`1044 sqft x 1.10 = 1148.40 sqft`

Then divide by product coverage and round up to full stock unit.

Examples from current NII price reference:

| Product | Coverage | Order qty | Note |
|---|---:|---:|---|
| 4.5mm SPC | 24.16 sqft/box | 48 boxes | 1148.40 / 24.16 = 47.53, round to 48. |
| 3mm Vinyl | 36 sqft/box | 32 boxes | 1148.40 / 36 = 31.90, round to 32. |
| Carpet | Use roll/box coverage from product folder | Round up full roll/box | Do not use loose sqft unless supplier sells that way. |

## Skirting rule

Skirting is not always needed for every floor covering.

- SPC: skirting is normally considered unless existing skirting remains or owner excludes it.
- Vinyl: skirting depends on existing wall edge/detail.
- Carpet: skirting is usually not added by default unless product/system or owner request says so.

Calculation when skirting is included:

| Formula | Result |
|---|---:|
| Perimeter | 129.24 ft |
| Add 10% wastage | 142.17 ft |
| Divide by stock length | Use actual length from price list |
| Round up | Full pcs only |

If skirting stock length is 8ft:

`142.17 / 8 = 17.77 -> 18 pcs`

If skirting stock length is 9ft:

`142.17 / 9 = 15.80 -> 16 pcs`

## Profile / trims rule

Most floor coverings need profile/trims at doors, transitions, ends or level changes.

- Carpet usually uses profile/trims at door transitions and exposed edges.
- Vinyl usually uses profile/trims at door transitions and ends.
- SPC usually uses profile/trims at door transitions, end caps, reducer/threshold, or expansion joints.

Use actual door/opening count if known.

If not known, use logical standard allowance for house size.

For 1044 sqft example:

| Formula | Result |
|---|---:|
| Door/opening allowance | 8 openings |
| Standard door width | 3 ft/opening |
| Total profile length | 24 ft |
| Add 10% wastage | 26.40 ft |
| Profile stock length example | 9 ft/pcs |
| Order qty | 26.40 / 9 = 2.93 -> 3 pcs |

## Foam / underlay rule

SPC:

- Ground floor normally uses foam-backed / foam-included SPC when available.
- If selected SPC product includes foam/underlay, record `FOAM INCLUDED IN PRODUCT SPEC` and do not add separate foam cost.
- Upper floor normally has no separate foam by default unless product spec, acoustic requirement or site requirement says so.
- If separate foam is required, calculate by supplier roll/box size only.

Vinyl:

- Foam/underlay is not added unless product/system specifically requires it.
- Glue-down vinyl normally uses glue, not loose foam.

Carpet:

- Carpet may use underlay/foam depending on product/system and customer selection.
- If underlay is required, calculate by supplier roll/stock size, not loose small sqft.

## Glue rule

SPC:

- Click-system SPC normally does not need full-area glue.
- Include glue only for profile, edge/detail or selected system if required.

Vinyl:

- Glue-down vinyl normally needs glue.
- Use stock-size tub allowance if exact coverage is not listed.

Carpet:

- Carpet may require adhesive, tape or gripper/profile depending on system.
- Use the product folder / installer method to decide whether glue is required.

## Labour rule

Use the job labour rate given by owner.

Example:

`1044 sqft x RM1.20/sqft = RM1,252.80`

State whether labour includes delivery, handling, installation, profile fixing or only installation.

## Output requirement

Each calculation should show:

1. Product type: Carpet / Vinyl / SPC.
2. Floor level: ground floor / upper floor / mixed.
3. Area used.
4. Wastage percentage.
5. Product coverage or stock size.
6. Rounded order quantity.
7. Accessories: skirting, profile, glue, foam/underlay.
8. Labour rate.
9. Working cost subtotal.
10. Whether markup is included or excluded.

## Important warning

This is a cost-estimate template. It is not customer selling price unless markup/margin is calculated separately.
