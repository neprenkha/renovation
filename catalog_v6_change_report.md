# Catalog V6 Change Report

## Purpose

V6 is created as a new versioned file set so earlier catalog versions remain intact. It combines the previous catalog price references, customer-facing scope lock, latest Materials/Labour references and owner decisions into a more complete public guide.

## Files added in V6

- `single_storey_house_extension_public_catalog_v6.md`
- `double_storey_house_extension_public_catalog_v6.md`
- `catalog_v6_customer_answer_sheet.md`
- `catalog_v6_change_report.md`

## Versioning rule followed

Older catalog files were not overwritten, deleted, renamed or moved. V6 is a new latest version. If V6 is corrected later, the next major public update should create V7 instead of overwriting V6.

## Sources considered

- Existing V4 public catalog prices and exclusions.
- Prior V5 customer-facing scope decisions.
- `catalog_v5_customer_scope_lock.md`.
- `catalog_versioning_workflow.md`.
- `1.2 Materials.txt` for material master list availability.
- `1.3 Labour.txt` for wiring, installation, skilled worker, helper, rental and disposal references.
- Deep Research package intent where not conflicting with the above.

## Customer-facing improvements

V6 now makes it easier to answer customer questions without naming brands:

- floor tile category by tier;
- tile skirting inclusion;
- plaster ceiling board and design ceiling rule;
- door and window allowance by approximate size;
- electrical point allowance by package;
- plumbing allowance by package and work area;
- bathroom/WC tier differences;
- toilet door size allowance;
- wet kitchen sink/tap/table-top scope;
- cabinet and premium top exclusions;
- paint system basis;
- double-storey structure and upper-floor plumbing notes.

## Price preservation

The following guide values were carried forward without recalculation:

- single-storey back/kitchen/yard ready prices;
- single-storey front/porch ready prices;
- single-storey size-band rates and large-extension thresholds;
- double-storey ground-floor, upper-floor, front/porch and full ground+upper guide prices;
- wet kitchen/table-top guide prices;
- bathroom add-on guide prices;
- laundry, electrical and plumbing add-on guide prices;
- custom-quote thresholds.

## Brand rule applied

Public wording does not list supplier or brand names. It describes item type, size allowance, material tier, quantity and exclusions. Brand/model is reserved for the official quotation.
