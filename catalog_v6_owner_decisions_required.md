# Catalog V6 Owner Decisions Required

V6 is usable as the latest public guide because it answers customer questions by item specification, size allowance, material tier and included quantity without naming brands.

The items below are not blockers. They are optional locks if the owner wants V7 to be stricter and closer to a fixed quotation template.

| # | Item | Current V6 treatment | Optional owner lock for future V7 |
|---:|---|---|---|
| 1 | Floor tile RM allowance | V6 states Basic economy ceramic, Standard better ceramic/entry vitrified, Better upgraded vitrified/premium-category. | Confirm exact RM material allowance and common tile size range by tier. |
| 2 | Bathroom tile RM allowance | V6 states bathroom tile tier by package and wall tile to agreed height. | Confirm floor/wall tile RM allowance and exact wall height rule. |
| 3 | Toilet door default | V6 includes toilet door set up to approx. 2.5ft x 7ft. | Confirm default material such as PVC or aluminium/PVC and allowance. |
| 4 | Extension door default | V6 includes one standard single-leaf door set up to approx. 3ft x 7ft where layout requires it. | Confirm default material by work area, for example back/kitchen/yard versus room/front. |
| 5 | Aluminium window allowance | V6 includes one aluminium framed window allowance by package size. | Confirm exact default frame system, glass type and cost allowance. |
| 6 | Sink/tap allowance | V6 allows standard sink/tap in Wet Kitchen Set A/B/C when listed. | Confirm standard sink/tap allowance and premium upgrade trigger. |
| 7 | Paint range | V6 states sealer/primer where required and two coats paint, with premium range for Better where listed. | Confirm paint range/tier names without publishing brand unless quotation lists it. |
| 8 | Light fittings | V6 includes light points by tier, not fixture model unless listed. | Confirm whether packages include actual fittings or only points. |

## Rule for current quotations

When preparing a quotation, list the exact selected item size, quantity, model/tier and any upgrade charge. Do not promise brand/model verbally if it is not written in the quotation.
